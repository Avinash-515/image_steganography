# frontend-react

This is a Create-React-App style scaffold for the existing frontend.

Run locally:

```bash
# install
npm install

# start dev server
npm start
```

By default the app uses relative `/api/*` URLs. To point to a different backend set `REACT_APP_API_BASE` before starting, e.g.: `REACT_APP_API_BASE=http://localhost:5000 npm start`.
